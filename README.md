## 📌 Featured Project
### 🔍 Botnet Traffic Detection System
> Final Year Project — University of Greenwich

Built a machine learning pipeline to detect botnet traffic within real-world 
network flow data using the CTU-13 dataset.

- Trained and compared Random Forest, XGBoost and Logistic Regression classifiers
- Achieved 0.99 accuracy and 0.81 F1-score on botnet detection
- Integrated SHAP explainability to identify key threat indicators
- **Tools:** Python · scikit-learn · XGBoost · SHAP · pandas · Matplotlib
